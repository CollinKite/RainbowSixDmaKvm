# Updated vmread for Windows 20H2 
# Building
  ```bash
cd vmread  
meson ../build
cd ../build
ninja
  ```
# Features
* Cav esp
* No recoil
* No spread
* Run and shoot
* Unlock all
* Glow
* Player fov (commented in code)
* No flash
* No aim animation
* Firing mode auto on all weapons

# Changing config
* Open config.h with text editor
* Change defines as you wish
